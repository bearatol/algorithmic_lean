## Рекомендуемый график

### Неделя 1: Array / String / Two Pointers (10 задач)
Начните с простых задач на массивы и строки. Паттерн Two Pointers используется в задачах 6,7.

### Неделя 2: Sliding Window / Prefix Sum / Hash Map (8 задач)
Sliding Window - задачи 11-14. Hash Map - задачи 15-18.

### Неделя 3: Stack / Queue / Linked List (6 задач)
Stack задачи 19-21. Queue - 22-23. Linked List - 24.

### Неделя 4: Binary Tree / BST (8 задач)
DFS/BFS обходы, поиск в BST.

### Неделя 5: Graph / Heap (5 задач)
Graph BFS/DFS, Priority Queue задачи.

### Неделя 6: Binary Search / Backtracking (5 задач)
Binary Search - 38-40. Backtracking - 41-42.

### Неделя 7: Dynamic Programming (8 задач)
1D DP - 43-46. 2D DP - 47-50.

### Неделя 8: Bit / Trie / Intervals / Monotonic Stack (15 задач)
Bit Manipulation - 51-53. Trie - 54-55. Intervals - 56-57. Monotonic Stack - 58-59. Дополнительные - 60-65.
